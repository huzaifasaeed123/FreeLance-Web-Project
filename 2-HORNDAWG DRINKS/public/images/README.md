# Image Assets

Place the following images in this directory:

## Required Images:

### Logo
- `logo.png` - Main Horn Dawg logo

### Home Page
- `background.png` - Hero section background
- `flavour_image1.png` - First flavour product
- `flavour_image2.png` - Second flavour product
- `flavour_image3.png` - Third flavour product
- `flavour_image4.png` - Fourth flavour product
- `flavour_2nd_image1.png` - Product image 1
- `flavour_2nd_image2.png` - Product image 2
- `flavour_2nd_image3.png` - Product image 3

### About Page
- `about_background.png` - About page hero background
- `about_image1.png` - About gallery image 1
- `about_image2.png` - About gallery image 2
- `about_image3.png` - About gallery image 3

## Image Specifications:

- Format: PNG or WebP (PNG recommended for logo)
- Logo: Transparent background, 500x500px minimum
- Backgrounds: 1920x1080px minimum
- Product images: 800x800px minimum
- Optimize all images before uploading

## Tools for Image Optimization:
- TinyPNG (https://tinypng.com/)
- ImageOptim (Mac)
- Squoosh (https://squoosh.app/)
