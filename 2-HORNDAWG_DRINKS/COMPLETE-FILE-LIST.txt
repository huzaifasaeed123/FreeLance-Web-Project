â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•

1. Copy your 5 HTML files to parent directory:
   - index.html (or index__2_.html)
   - about.html
   - contact.html
   - reservation.html
   - thank-you.html

2. Run conversion script:
   $ node scripts/convert-html-to-ejs.js

   $ npm install

   $ npm run dev
